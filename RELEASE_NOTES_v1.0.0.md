# MacroApp v1.0.0

Phiên bản phát hành đầu tiên của MacroApp cho Windows.

## Có gì trong bản này

- Ghi macro bàn phím và chuột theo thời gian thực
- Ghi thao tác giữ và thả phím
- Ghi quỹ đạo di chuyển chuột
- Phát macro theo thứ tự đã thiết lập
- Hỗ trợ lặp vô hạn
- Dừng an toàn bằng `F8`
- Timeline editor để chỉnh sửa từng action
- Bản dựng sẵn `StudioMacro.exe` cho Windows

- `StudioMacro.exe`
